Hi, Welcome to Employee REST API App (Mybatis)

Please Use Chrome Extension: "tabbed postman rest client"

1) Create an employee

localhost:8080/POST/employee

Test Data:
{ "empName": "Vignesh S", "empAge": 51, "profession": "Director" }

2) View All employees

localhost:8080/GET/employee

3) View an employee with ID

localhost:8080/GET/employee/{id}

4) View employees with IDs

localhost:8080/GET/employee/ids/{ids}

5) Update an employee

localhost:8080/PUT/employee

Test Data:
{ "postId": 1, "empName": "Marigold", "empAge": 25, "profession": "GM" }

6) Delete an employee with ID

localhost:8080/DELETE/employee/id

7) Delete employees with IDs

localhost:8080/DELETE/employee/ids/{ids}
